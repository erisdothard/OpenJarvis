Chromatic Liquid Metal

Cutting-edge CSS techniques for iridescent spectral dispersion, chromatic aberration, and liquid metal effects. All modern CSS, zero JavaScript.

1. Conic-Gradient Spectral Border

Uses CSS conic-gradient to create a 360° spectral wrap that rotates. Looks like light refracting through a prism.

2. Void-to-Prism Card (Black at Rest, Chromatic on Hover)

Card emerges from pure black void. No visible border at rest. On hover, chromatic prismatic edges appear like light catching the surface.

Live Demo
Hover me to see prismatic edges emerge
Usage
<div class="chromatic-void-card"> Content appears in void </div>
Technical Details

Uses ::before for spectral border (masked to border only), ::after for inner glow. Border opacity: 0 at rest, animates on hover. Gradient angle: 135° for diagonal prism effect.

3. Chromatic Ambient Glow (Background)

Subtle, animated radial glows in background. Shifts slowly like nebula. Three layers: cyan (top-left), violet (bottom-right), magenta (center).

✓ Fixed position, doesn't interfere with content (pointer-events: none)
✓ Three animated layers: ellipse at different positions and rotations
✓ 20-second loop for slow, ambient feel
✓ Alpha: 0.01–0.06 for ultra-subtle effect

CSS Features Used:
→ @property — Animatable custom properties (enables smooth color transitions)
→ conic-gradient() — 360° radial color wheels (prism effect)
→ mask-composite: exclude — Border-only masking (no fill)
→ radial-gradient() — Multi-layer ambient glows
→ ::before / ::after — Pseudo-elements for layering
→ mix-blend-mode — Color channel blending
→ filter: hue-rotate() — Spectral shift animation
→ cubic-bezier() — Easing: cubic-bezier(0.16, 1, 0.3, 1) for snappy feel
